# Simple Sticky Notes Issues

## Project Description
This project is a simple sticky notes application that allows users to create, manage, and delete sticky notes on their digital workspace.

## Documentation
### Features
- Create new sticky notes
- Edit existing notes
- Delete notes
- Drag and drop notes to arrange them

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dmi-ssn/simple-sticky-notes-issues.git
   ```
2. Navigate to the project directory:
   ```bash
   cd simple-sticky-notes-issues
   ```
3. Open the index.html file in your browser to view the application.

### Usage
- Click on the '+' button to create a new sticky note.
- Double-click on an existing note to edit it.
- Click the 'X' button to delete a note.
